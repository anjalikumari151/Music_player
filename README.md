# 🎵 Modern Music Player

A modern and responsive Music Player built using **HTML**, **CSS**, and **JavaScript**.

## ✨ Features

- ▶ Play / Pause Music
- ⏭ Next Song
- ⏮ Previous Song
- 🎵 Active Song Highlight
- 🖼 Dynamic Song Cover Image
- ⏱ Music Duration & Current Time
- 🌊 Animated Sound Wave
- 📱 Modern UI Design
- 🎧 Multiple Song Support

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🚀 How to Run

1. Download or Clone the Repository

```bash
git clone https://github.com/your-username/music-player.git
```

2. Open the project folder

3. Run `index.html` in your browser

## 🎶 Add Your Own Songs

Replace:

```javascript
src:"song1.mp3"
```

with your own music file.

Example:

```javascript
src:"music/faded.mp3"
```
